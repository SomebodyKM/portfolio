> **A Gamified Developer Portfolio**
>
> Welcome to my digital playground. This portfolio reimagines the standard "About Me" page as an immersive RPG character profile, featuring a quest log for projects, a skill tree for technologies, and a guild contact system.

---

## üéÆ Features

- **Immersive UI**: Cyberpunk/Sci-Fi aesthetic with neon glows, glitch effects, and scanlines.
- **Player Profile**: Dynamic leveling system where XP is calculated based on completed projects.
- **Quest Log**: A filterable project showcase (Completed, In-Progress, Planned) with difficulty ratings and XP rewards.
- **Skill Tree**: Visual representation of technical proficiency (HTML, CSS, React, Node.js, etc.).
- **Contact Guild**: Fully functional contact form powered by **EmailJS** with spam protection and rate limiting.
- **Responsive Design**: Optimized for mobile, tablet, and desktop (Desktop recommended for full immersion).
- **Animations**: Custom CSS keyframes for particles, typing effects, and page transitions (No heavy animation libraries).
